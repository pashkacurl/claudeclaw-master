# 🎯 Creator Marketing - Маркетолог для контент-криейтора

## 📦 Установка

1. Скопируйте папку `creator-marketing/` в `skills/` вашего workspace
2. OpenClaw автоматически подхватит скилл при следующем сообщении
3. Проверка: спросите агента "помоги с маркетингом канала"

## 🎯 Что умеет

- Создавать заголовки для YouTube с высоким CTR
- Писать продающие посты для Telegram
- Разрабатывать воронки free → paid
- Рассчитывать ценообразование и стратегию монетизации
- Планировать запуски продуктов
- Применять психологию маркетинга и триггеры конверсии
- Редактировать тексты для максимальной вовлеченности

## ⚙️ Как работает

### Основные модули

#### 1. 🎬 YouTube

**Формулы заголовков:**
- **Числа + Обещание:** "5 инструментов, которые заменят вам команду"
- **Как + Результат:** "Как я автоматизировал 80% рутины за выходные"
- **Противопоставление:** "A vs B: что выбрать в 2026"
- **История:** "Я месяц жил с [X]. Вот что вышло"
- **Провокация:** "Почему ваш [инструмент] - это игрушка"

**CTR-оптимизация:**
- Заголовок: 50-60 символов, ключевое слово в начале
- Превью: лицо + крупный текст (3-4 слова) + контрастные цвета
- Первые 30 секунд: hook → обещание → "досмотри до конца"
- Удержание: менять темп каждые 2-3 минуты

**Описания:**
- Первые 2 строки - SEO + интрига (видны до "показать ещё")
- Тайм-коды обязательно
- CTA: подписка на Telegram → платный канал
- Хештеги: 3-5 релевантных

#### 2. 📱 Telegram

**Форматы постов (от лучших к худшим):**
1. **Кейс с цифрами** - "Настроил X за 2 часа, сэкономил 15 часов в неделю"
2. **Сравнение/батл** - "A vs B: мой опыт за месяц"
3. **Личная история + вывод** - "Почему я перешёл с [X] на [Y]"
4. **Мини-гайд** - "3 шага чтобы [результат]"
5. **Мнение/философия** - "[Тренд] не убивает [индустрию]"
6. **Новость + разбор** - Не просто "вышло обновление", а "что это значит для вас"

**Структура поста:**
```
🔥 Хук (1 строка - зацепить)

Контекст (2-3 предложения - почему это важно)

Мясо (основная ценность - факты, цифры, опыт)

💡 Вывод / CTA (1-2 строки)
```

**Воронка: бесплатный → платный:**
1. YouTube видео → CTA "подробнее в Telegram"
2. Telegram пост → ценность + упоминание "в платной группе разобрал детально"
3. Платная группа → эксклюзив (готовые решения, конфиги, разборы)

#### 3. 💰 Монетизация и ценообразование

**Психология ценообразования:**
- **Якорение:** покажи ценность в часах ("этот инструмент сэкономит 10 часов/мес, стоит как чашка кофе")
- **Фрейминг потерь:** "каждый день без X - Y часов впустую"
- **Социальное доказательство:** "уже N человек внутри"
- **Цена через ценность:** не "5€/мес за контент", а "5€/мес за готовые решения"

**Стратегия роста цены:**
1. Старт: 5€/мес (набор первых 50)
2. При 50: поднять до 7€ ("для первых 50 цена заморожена")
3. При 100: ввести годовой план со скидкой
4. При 200+: тиры (базовый / про с консультациями)

#### 4. 🚀 Запуск продуктов

**Формула запуска (7 дней):**
| День | Действие |
|------|----------|
| -7 | Тизер: "Готовлю кое-что..." |
| -5 | Проблема: пост о боли, которую решает продукт |
| -3 | Социальное доказательство: бета-тестеры, скриншоты |
| -1 | Countdown: "Завтра открываю доступ" |
| 0 | ЗАПУСК: подробный пост + видео + CTA |
| +1 | Отзывы первых покупателей |
| +3 | "Осталось X мест" / повышение цены |

**Для YouTube-видео:**
1. Анонс за 3-5 дней в Telegram
2. Premiere вместо обычной публикации
3. Пост в Telegram сразу после выхода
4. Stories/repost через 24 часа

#### 5. 🧠 Психология маркетинга

| Модель | Применение |
|--------|-----------|
| **Jobs to Be Done** | Люди покупают результат, не процесс. "Научиться X" → "освободить 10ч/нед" |
| **Парето 80/20** | 20% контента даёт 80% подписчиков. Найди хиты и делай больше |
| **Якорение** | Сначала покажи полную ценность, потом цену |
| **FOMO** | Ограниченные места, повышение цены |
| **Reciprocity** | Дай бесплатно → человек "должен" → покупает |
| **Social Proof** | Отзывы, числа, скриншоты результатов |
| **Authority** | Экспертизу через кейсы, не дипломы |
| **Scarcity** | "Первым 20 - бонус" (реальный!) |
| **Commitment** | Маленький шаг → большой. Подписка → покупка |

#### 6. ✍️ Копирайтинг

**Принципы:**
- **Ясность > креативность** - если непонятно, значит плохо
- **Выгоды > фичи** - не "AI с RAG", а "ответит на любой вопрос по вашим документам"
- **Одно сообщение = одна мысль**
- **CTA конкретный** - не "узнать больше", а "скачай шаблон"

**Редактирование (чеклист):**
1. Убери первый абзац (часто разгон)
2. Замени пассивный залог на активный
3. Убери слова-паразиты: "просто", "на самом деле", "в целом"
4. Каждое предложение - одна мысль
5. Прочитай вслух - спотыкаешься? Перепиши

#### 7. 📧 Воронка для платных подписчиков

**Welcome-серия (когда человек подписался):**
| # | Когда | Тема | Цель |
|---|-------|------|------|
| 1 | Сразу | "Добро пожаловать! Вот что тебя ждёт" | Ориентация |
| 2 | День 2 | Лучший материал (wow-эффект) | Подтвердить ценность |
| 3 | День 5 | "Вот что сделали другие участники" | Social proof |
| 4 | День 14 | "Как дела? Что попробовал?" | Вовлечение |

### Правила текста

> **Стиль и антиканцелярит → используй скил `copywriter`.** Там полный список запрещённых слов, формат постов, метафоры и паттерны. Этот скил фокусируется на стратегии, не на стиле.

### Workflow

1. **Определи задачу:** контент / монетизация / запуск / стратегия
2. **Учитывай аудиторию:** новички или профи? B2B или B2C?
3. **Давай конкретику:** не "напиши хороший пост", а готовый текст
4. **Проверяй:** есть ли психологические триггеры? CTA? Понятно ли новичку?

## 🔧 Настройка под себя

### Для разных ниш

**Tech/AI:**
- Фокус на практику и кейсы
- Цифры важнее слов
- Аудитория ценит честность

**Образование:**
- Больше историй успеха
- Социальное доказательство критично
- CTA через бесплатный вебинар

**B2B:**
- ROI и экономия времени/денег
- Кейсы с конкретными компаниями
- Длинные циклы продаж

**Личный бренд:**
- Authenticity > продажи
- История и путь важнее результата
- Community-driven монетизация

### Настройка стиля

**Формальный (B2B):**
- Уберите мат
- Добавьте больше данных
- CTA через консультацию

**Неформальный (комьюнити):**
- Можно материться
- Больше юмора и личного
- CTA через общение

### Частота постов

- **YouTube:** 1-2 раза в неделю (качество > количество)
- **Telegram:** 3-5 раз в неделю (присутствие)
- **Платная группа:** ежедневно (ценность за подписку)

### Аналитика

Отслеживайте:
- CTR заголовков (меняйте формулы)
- Удержание видео (где отваливаются?)
- ERR постов (какие темы заходят?)
- Конверсия free → paid (где теряете людей?)
