# Competitor Monitor - Мониторинг конкурентов

## Когда использовать

- "Следи за конкурентами"
- "Что изменилось у [конкурент]?"
- "Сравни нас с конкурентами"
- "Мониторинг цен / контента / позиций"

## Типы мониторинга

### 1. Snapshot сайта (разовая проверка)

```bash
# Скачать страницу для сравнения
curl -s "https://competitor.com/pricing" -o /tmp/competitor-pricing-$(date +%Y%m%d).html

# Извлечь текст
python3 -c "
from html.parser import HTMLParser
class P(HTMLParser):
    text = []
    def handle_data(self, d): self.text.append(d.strip())
p = P()
p.feed(open('/tmp/competitor-pricing-$(date +%Y%m%d).html').read())
print('\n'.join(t for t in p.text if t))
"
```

### 2. Diff изменений (что поменялось)

```bash
# Сравнить две версии
diff /tmp/competitor-pricing-20260410.html /tmp/competitor-pricing-20260417.html
```

### 3. Мониторинг через web_search

```
web_search("[конкурент] новости OR обновления OR запуск", freshness="pw")
web_search("[конкурент] pricing OR цены", freshness="pm")
web_search("site:[конкурент].com", freshness="pw")
```

## Что отслеживать

### Сайт и продукт
- [ ] Цены (изменения, новые тарифы)
- [ ] Фичи (новые, удалённые)
- [ ] Лендинг (текст, CTA, дизайн)
- [ ] Блог (новые статьи, темы)
- [ ] Вакансии (показывают направление роста)

### Контент и соцсети
- [ ] YouTube (новые видео, темы, частота)
- [ ] Telegram (посты, подписчики, ERR)
- [ ] Instagram (посты, рилсы, подписчики)
- [ ] Twitter/X (активность, темы)

### SEO
- [ ] Позиции по ключевым запросам
- [ ] Новые страницы (site: оператор)
- [ ] Обратные ссылки (через ahrefs/dataforseo)

### Бизнес
- [ ] Новые клиенты (кейсы, отзывы)
- [ ] Партнёрства (интеграции, коллабы)
- [ ] Инвестиции / раунды (crunchbase, новости)

## Формат отчёта

```markdown
# Мониторинг конкурентов — [Период]

**Дата:** YYYY-MM-DD
**Конкуренты:** [список]

## Ключевые изменения

### [Конкурент 1]
- **Цены:** [изменились / не изменились]
- **Продукт:** [новые фичи / обновления]
- **Контент:** [новые статьи / видео]
- **Вывод:** [что это значит для нас]

### [Конкурент 2]
...

## Рекомендации
1. [Что нам стоит сделать в ответ]
2. [Возможность которую они упускают]
3. [Угроза на которую обратить внимание]

## Источники
- [url1] — что нашли
- [url2] — что нашли
```

## Автоматизация (через cron)

Для регулярного мониторинга можно настроить:

```bash
# Еженедельный мониторинг (понедельник 9:00)
node /home/aibot/claudeclaw-BANDA/dist/schedule-cli.js create \
  "Проведи мониторинг конкурентов [список]. Что изменилось за неделю: цены, фичи, контент. Отправь отчёт." \
  "0 9 * * 1"
```
