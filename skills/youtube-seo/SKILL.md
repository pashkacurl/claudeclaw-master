# YouTube SEO Generator - Оптимизация для поиска

## 📦 Установка

1. Скопируйте папку `youtube-seo/` в `skills/` вашего workspace
2. OpenClaw автоматически подхватит скилл при следующем сообщении
3. Проверка: отправьте агенту транскрипт видео или аудиофайл

## 🎯 Что умеет

- Генерировать кликабельные названия с учетом SEO
- Писать описания с естественной интеграцией ключевых слов
- Создавать структурированные тайм-коды
- Подбирать хештеги для описания и загрузки
- Анализировать транскрипты и выделять главное

## ⚙️ Как работает

### Workflow

**Шаг 1: Получение транскрипта**

Если у вас есть видео/аудио:
```bash
# Извлечь аудио (если нужно)
ffmpeg -i video.mp4 -vn -acodec libmp3lame audio.mp3

# Транскрибировать через Whisper API
# (нужен скилл openai-whisper-api)
```

Если есть готовый транскрипт - используйте напрямую.

**Шаг 2: Анализ контента**

1. Определить главную тему и цель видео
2. Извлечь ключевые инсайты, цифры, факты
3. Понять структуру повествования (для тайм-кодов)
4. Определить тип контента:
   - Туториал → фокус на ценность и шаги
   - Кейс/исследование → фокус на результаты
   - Обзор/сравнение → фокус на выбор
   - Интервью → фокус на эксперта и мнение

**Шаг 3: Генерация метаданных**

### Правила написания

**Стиль:**
- От первого лица (как автор канала)
- Тон: искренний, профессиональный, не навязчивый
- Максимум 1 эмодзи во всём описании
- Избегать клише: "В этом видео я расскажу..."
- Естественная интеграция ключевых слов, без спама

**Запрещено:**
- Длинное тире (—), используй дефис (-)
- Канцелярит: "является", "осуществляет", "обеспечивает"
- Навязчивые призывы: "жми лайк/подпишись"
- Кликбейт и обман ожиданий
- Триггеры демонетизации YouTube (насилие, финансовые пирамиды, мат)

### Требования к названию

- **70-75 символов оптимально**, максимум 100
- Первые 70 символов = самая цепляющая часть
- Включить 1-2 ключевых слова
- Использовать цифры где уместно ("7 принципов...")
- Интрига + конкретика + ценность

**Формулы названий:**
- "Как [действие] за [время/ресурс]"
- "[Число] способов [результат]"
- "[Сравнение]: что выбрать в [год]"
- "Почему [утверждение] и как это изменит [область]"

Каждый вариант с обоснованием: SEO-ценность, кликабельность, эмоциональный триггер.

### Требования к описанию

**Структура (по порядку):**
1. **Хук** - мощное первое предложение, БЕЗ "В этом видео..."
2. **Что узнает зритель** - ключевые инсайты, методы, конкретная ценность
3. **Блоки тем** - структурированный обзор содержания
4. **Мягкий CTA** - ненавязчивый призыв
5. **Ссылка на сообщество** (если есть Telegram/Discord)
6. **Тайм-коды**
7. **Хештеги для описания**

**Первые 2 строки критичны** - видны до "показать ещё". Вся интрига здесь.

### Требования к тайм-кодам

Формат: `ММ:СС - Название секции`

- Основаны на реальном контенте транскрипта
- Четкие, информативные названия секций
- 8-15 секций в зависимости от длины
- Первый: `0:00 - Введение` или аналог

Пример:
```
0:00 - Введение
1:45 - Проблема которую решаем
5:20 - Настройка окружения
12:30 - Основные шаги
18:15 - Типичные ошибки
22:40 - Результаты и выводы
```

### Требования к хештегам

**Хештеги для описания:**
- Формат: `#хештег1 #хештег2 ...`
- Брендовые (если есть): `#ВашБренд`
- Тематические на основе контента
- **15-20 хештегов всего**
- Смешанный язык (русский + английский для охвата)

Пример:
```
#искусственныйинтеллект #ai #автоматизация #programming 
#tutorial #нейросети #aiagent #promptengineering
```

**Хештеги для загрузки (upload tags):**
- Формат: через запятую, **БЕЗ символа #**
- **Максимум 500 символов**
- Могут пересекаться с хештегами описания
- Добавить средне- и низкочастотные запросы
- Смешанный язык

Пример:
```
искусственный интеллект, автоматизация, AI, программирование, 
tutorial, нейросети, prompt engineering, ChatGPT, для начинающих
```

## 💡 Примеры использования

**Пример 1: Туториал**

```
Отправляете транскрипт видео про автоматизацию задач.

Агент генерирует:

   (личный результат + цифры)

✅ Рекомендуемое:
"Автоматизация рутины за 30 минут - пошаговый гайд"

📝 Описание:
Большинство тратит часы на задачи которые можно автоматизировать 
за 30 минут. В этом гайде покажу как.

Вы узнаете:
- Какие задачи автоматизировать в первую очередь
- Инструменты которые не требуют программирования
- Готовые шаблоны для старта

⏱ Тайм-коды:
0:00 - Введение
2:15 - Выбор задач для автоматизации
7:30 - Настройка первого сценария
...

#автоматизация #productivity #tutorial
```

**Пример 2: Обзор/Сравнение**

```

"Tool A vs Tool B: честное сравнение после 3 месяцев"
```

## 🔧 Настройка под себя

### Адаптация под нишу

**Техническая аудитория:**
- Больше терминов и конкретики
- Меньше общих слов
- Хештеги: `#programming #development #tech`

**Массовая аудитория:**
- Простой язык
- Больше аналогий
- Хештеги: `#для_начинающих #простыми_словами`

**Бизнес-контент:**
- Фокус на ROI и практическую пользу
- Кейсы и цифры
- Хештеги: `#бизнес #маркетинг #продажи`

### Язык контента

**Русскоязычный канал:**
- Основные хештеги на русском
- Английские только популярные (#AI, #tutorial)
- Ключевые слова в названии на русском

**Англоязычный канал:**
- Всё на английском
- Keywords: automation, AI, productivity, tutorial

**Смешанный:**
- Название на основном языке аудитории
- Хештеги 50/50
- Описание на основном языке + краткий перевод

### Тип канала

**Образовательный:**
- Фокус на "научитесь", "узнаете", "пошагово"
- Тайм-коды детальные
- CTA: "попробуйте и напишите результат"

**Развлекательный:**
- Эмоциональные названия
- Меньше структуры, больше энергии
- CTA: "ставьте лайк если согласны"

**Обзорный:**
- Сравнения и топы
- Объективность в описании
- CTA: "какой вариант выберете вы?"
