# Copywriter - Живая речь без канцелярита

## 📦 Установка

1. Скопируйте папку `copywriter/` в `skills/` вашего workspace
2. OpenClaw автоматически подхватит скилл при следующем сообщении
3. Проверка: спросите агента "напиши пост про..."

## 🎯 Что умеет

- Писать посты для Telegram в энергичном стиле
- Создавать описания для YouTube с тайм-кодами
- Генерировать анонсы и тизеры
- Выражать мнения и разбирать кейсы
- Избегать AI-канцелярита и шаблонных фраз

## ⚙️ Как работает

### Стиль подачи

**Тон:**
- Энергичный, прямой, дружелюбный
- Обращение к аудитории создает личный диалог
- Юмор и самоирония делают контент живым
- Уверенная экспертная позиция

**Эмоциональность:**
- Минимум эмодзи (1-2 на пост максимум)
- Восклицательные знаки для драйва (без фанатизма)
- Жирный текст для ключевых идей

**Фокус:**
- Практическая польза
- Решение конкретных проблем
- Собственный опыт и инструменты

**Структура:**
- Короткие ударные фразы чередуются с развернутыми
- Абзацы разной длины (рваный ритм)
- Никаких маркированных списков без необходимости

**Язык:**
- Технически грамотный, но доступный
- Сильные глаголы действия
- Метафоры и яркие аналогии
- Разговорные выражения для близости

### АНТИ-AI-ИЗМЫ (обязательно)

**Строго запрещено:**
- "стоит отметить", "важно понимать", "более того"
- "является", "обеспечивает", "осуществляет"
- "в современном мире", "на сегодняшний день"
- "ключевой", "спектр", "погрузиться"
- "в заключение", "безусловно", "конечно"
- "однако" (в начале предложения)
- "давайте рассмотрим", "не секрет что"
- "в рамках", "в контексте", "данный" (вместо "этот")

**Пунктуация:**
- Минимум длинных тире (—), используй дефис (-)
- Никаких двоеточий в заголовках
- Без вводных слов в начале предложений

**Глаголы:**
- ❌ "осуществляет проверку" → ✅ "проверяет"
- ❌ "является причиной" → ✅ "вызывает"
- ❌ "производит анализ" → ✅ "анализирует"
- ❌ "имеет возможность" → ✅ "может"

**Финал:**
Никогда не пиши выводы в стиле "В заключение...", "Важно помнить...", "Таким образом...". Просто обрывай мысль или заканчивай фактом.

### Типы контента

#### Пост в Telegram

**Структура:**
1. Хук - вопрос/факт/эмоция (без вводных слов!)
2. Контекст - что произошло
3. Детали (абзацами, не списками)
4. Вывод/призыв (без "В заключение")
5. Эмодзи для акцентов (1-2 максимум)

**Длина:** 300-800 знаков

**Шаблон:**
```
[Хук без вводных слов]

[Короткий абзац]

[Развёрнутое объяснение]

[Короткая мысль]

[Призыв к действию]

#хештеги
```

#### Описание YouTube

**Структура:**
1. Цепляющее название (30-60 знаков, без двоеточия)
2. Краткое описание (что узнаешь)
3. Тайм-коды (если длинное)
4. Ссылки на ресурсы
5. Призыв к действию

**Тон:** чуть более структурированный, но живой

#### Анонс

**Структура:**
1. Тизер (интрига)
2. Что будет
3. Для кого
4. Когда/где

### Характерные обороты

**Для объяснений:**
- "на пальцах", "давай разберем"
- "из коробки", "под капотом"
- "короче", "прикиньте"

**Для эмоций:**
- "это кайф", "щёлкнуло"
- "не могу остановиться"
- "снова удивили )))"

**Метафоры:**
- Инструменты → "тяжёлая артиллерия"
- Обновления → "пачка фиксов", "прошивка"
- Объяснение → электроинструменты vs IKEA

**Структурные паттерны:**

Открытие поста (хук):
- Факт без вводных: "Обновил систему до...", "Вышла новая версия..."
- Провокация: "X умирает?", "Зачем это нужно?"
- Личный опыт: "Три дня назад гордился... А сегодня выключил"

Переход к сути:
- "Вот почему это важно"
- "Что изменилось?"
- "В чём фишка"
- Без "Давайте рассмотрим", "Стоит отметить"

Объяснение сложного:
- Через аналогию: "X = молоток. Y = шуруповёрт. Оба нужны"
- Или бытовую: "как дать пистолет пьяной обезьяне"

Закрытие:
- ВСЕГДА вопрос к аудитории: "А вы как делаете?", "У вас что нового?"
- НИКОГДА "В заключение", "Подводя итог"

### Что НИКОГДА не делать

❌ Начинать с "Конечно", "Безусловно", "Важно понимать"
❌ Писать "стоит отметить", "является", "обеспечивает"
❌ Заканчивать "В заключение хочется сказать"
❌ Использовать длинные тире (—)
❌ Ставить двоеточия в заголовках
❌ Нагромождать эмодзи (больше 2)
❌ Писать маркированные списки без запроса
❌ Использовать пассивный залог

### Что ВСЕГДА делать

✅ Начинать с хука (вопрос/факт/эмоция)
✅ Использовать сильные глаголы
✅ Писать абзацами разной длины (рваный ритм)
✅ Выделять **жирным** ключевые мысли
✅ Вплетать личный опыт и юмор
✅ Заканчивать призывом/вопросом
✅ Метафоры и аналогии из жизни
✅ Обращаться к аудитории

## 💡 Примеры использования

**Пример 1: Пост про обновление**
```
"Обновил систему до новой версии

Самое жирное - автоматические бэкапы. Раньше нужно было 
руками запускать скрипт. Сейчас крутится каждую ночь без 
участия.

Память теперь не теряется после перезагрузки. Для тех кто 
ведёт длинные диалоги - критично.

И пачка мелких фиксов по стабильности.

А вы обновляетесь сразу или ждёте?"
```

**Пример 2: Объяснение сложной темы**
```
"Зачем запускать агента локально, если есть облако за 5 баксов?

Разница не в деньгах. Разница в том, где живёт мозг твоего 
помощника.

Облако = отель. Удобно, но не твоё. Локально = свой дом. 
Свои правила.

Я за второе. Потому что мои данные - моя ответственность."
```

**Пример 3: Мнение про тренд**
```
"AI-агенты заменят разработчиков?

Короче, нет.

Агент = помощник, не замена. Как электроинструмент для 
плотника. Без навыков плотника инструмент бесполезен.

Оба нужны. Для разных задач."
```

## 🔧 Настройка под себя

### Измени стиль под свою аудиторию

**Если пишешь для бизнеса:**
- Убери жёсткие обороты ("хрен его знает")
- Оставь энергию и прямоту
- Добавь больше цифр и кейсов

**Если для разработчиков:**
- Можешь использовать технический жаргон
- Добавь примеры кода
- Меньше аналогий, больше конкретики

**Если для новичков:**
- Больше объяснений через аналогии
- Меньше терминов
- Пошаговые инструкции

### Настрой частоту эмодзи

По умолчанию: 1-2 на пост
- Для формальной аудитории: 0-1
- Для молодой аудитории: 2-3

### Длина контента

- **Короткие посты (300-500):** быстрая новость, мнение
- **Средние (500-800):** разбор темы, кейс
- **Длинные (800-1200):** глубокий анализ, инструкция

### Тематика

Скилл универсален. Работает для:
- Технологий и AI
- Бизнеса и стартапов
- Образования
- Личных блогов
- B2B и B2C

**Главное:** сохраняй живость, избегай канцелярита, говори с читателем как с другом.

### Разграничение с другими скиллами

- **copywriter** (этот скил) = ТЕКСТ. Стиль, формат, антиканцелярит, написание постов
- **creator-marketing** = СТРАТЕГИЯ. Воронки, ценообразование, запуски, психология продаж
- **youtube-seo** = SEO-метаданные. Заголовки, описания, теги, тайм-коды для YouTube
